Course Overview

What this course includes

Who it is for

Students, career switchers, designers, and beginner developers who want to create clean websites without relying only on templates.

Prerequisites

No prior coding experience is required. Basic computer skills and comfort using a browser are enough to begin.

Tools used

VS Code, browser developer tools, Git basics, responsive testing tools, and lightweight design references.

Final outcome

You will complete a responsive learning platform interface with course cards, forms, navigation, dark mode, and interactive JavaScript.

Learning Format

How you will learn

  • Short lessons with focused practice tasks
  • Weekly mentor checkpoints and code review notes
  • Downloadable starter files and project briefs
  • Final polish checklist before certificate submission

Curriculum

Course modules

01

HTML structure and semantic pages

Headings, sections, links, images, forms, tables, metadata, and accessible document structure.

02

CSS layout and visual design

Box model, flexbox, grid, responsive spacing, colors, typography, and reusable card systems.

03

JavaScript for interface behavior

DOM selection, events, menus, theme toggles, modal windows, validation, and simple page state.

04

Accessibility and responsive testing

Keyboard navigation, focus states, readable contrast, alt text, mobile checks, and reduced-motion support.

05

Deployment preparation

Folder structure, asset checks, link checks, browser testing, and final styling polish for launch.

06

Final portfolio project

Build and polish a responsive learning platform page with review feedback and a presentation walkthrough.
